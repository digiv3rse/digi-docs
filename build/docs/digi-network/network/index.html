Network Details

DiGi Network Mainnet#

DiGi Network is an L2 built on top of Ethereum using the OP stack.

NameValue
NetworkDiGi
Chain ID7777777
Gas TokenETH
RPChttps://rpc.digi.energy
Websocketwss://rpc.digi.energy
Block Explorerhttps://explorer.digi.energy

To get a non-rate-limited API key for the digi RPC, sign up for an account at Conduit.xyz for an RPC API key.
